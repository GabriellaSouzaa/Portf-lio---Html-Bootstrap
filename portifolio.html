Sou uma desenvolvedora front-end com conhecimentos em HTML, CSS e JavaScript. Embora ainda não tenha experiência profissional, estou motivada para iniciar na área. Principais Projetos

Projeto UPX: Monitoramento da Qualidade do Ar

Participei de um projeto em grupo na faculdade focado no monitoramento da qualidade do ar. Utilizamos um Raspberry Pi como microcontrolador, junto com um sensor de fumaça MQ-2 para detectar gases e fumaça. O sistema indicava a qualidade do ar por meio de LEDs (vermelho para ar poluído e verde para ar limpo) e enviava as informações em tempo real para uma API criada. Utilizamos Docker e PostgreSQL para armazenamento e tratamento dos dados.

Tela para controle de luminárias

Criei uma tela em HTML e CSS para a Faculdade Facens, onde havia um cursor deslizante para controlar a intensidade das luminárias e um botão de liga/desliga para poder controlar as luminárias da faculdade.
